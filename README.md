# Temu 补报活动自动处理脚本

> 自动完成 Temu 卖家平台商品补报活动的 SKU 批量处理，告别手动翻页点击。

## 功能

- 自动点击"一键填入参考价"填充价格
- 根据自定义成本价阈值（默认 ¥45）自动判断
- 全部 SKU 价格 >= 阈值 → 自动勾选并提交报名
- 存在低于阈值的 SKU → 自动跳过翻页
- 提交后弹窗自动回到第 1 页，继续遍历剩余页面
- 浮动控制面板，可拖拽、可调阈值、实时日志

## 适用场景

Temu 卖家平台的**商品补报活动**中，当有大量 SKU（如 279 条）需要逐一处理时，手动操作效率极低。本脚本自动完成：填价 → 判断 → 提交/跳过 的完整流程。

## 安装

### 1. 安装浏览器扩展

需要安装以下任一用户脚本管理器：

- [ScriptCat（脚本猫）](https://docs.scriptcat.org/docs/use/open-dev/?browser=chrome) — 推荐，国内访问快
- [Tampermonkey（油猴）](https://www.tampermonkey.net/)

### 2. 安装脚本

**方式一：通过脚本猫/油猴安装**

1. 点击 [temu-auto-supplement.user.js](./temu-auto-supplement.user.js)
2. 脚本管理器会自动弹出安装界面
3. 点击"安装"

**方式二：手动安装**

1. 复制 `temu-auto-supplement.user.js` 的全部代码
2. 打开脚本猫/油猴管理面板 → 新建脚本
3. 粘贴代码 → 保存

## 使用方法

1. 打开 Temu 卖家平台 → 营销活动 → 补报活动
2. 点击"补报提醒"打开 SKU 列表弹窗
3. 右上角会出现"Temu 补报自动处理"控制面板
4. 设置成本价阈值（默认 45 元）
5. 点击"▶ 开始执行"

脚本会自动遍历所有页面，低于阈值的页跳过，全部达标的页自动提交。

## 工作流程

```
打开补报弹窗
    ↓
点击"开始执行"
    ↓
┌→ 当前页填价（一键填入参考价）
│   ↓
│  检查价格是否全部 >= 阈值
│   ↓
│  ├─ 有低于阈值 → 跳过，翻下一页 → ┐
│  │                                  │
│  └─ 全部达标 → 勾选全选 → 确认报名  │
│       → 二次确认 → 提交成功         │
│       → 弹窗回到第1页 ─────────────┘
│
─ 一轮遍历无新提交 → 结束
```

## 注意事项

- 脚本仅在 `agentseller.temu.com` 下运行
- 成本价阈值可根据实际情况在控制面板中调整
- 提交后弹窗不会关闭，会自动回到第 1 页继续处理
- 建议先在少量页面测试，确认逻辑符合预期后再全量运行
- 脚本不会永久删除任何数据，仅模拟人工点击操作

## 技术说明

- 纯 JavaScript 实现，无外部依赖
- 通过 polling 机制等待 React 应用的状态更新
- 自动处理 Temu 平台的 CSS 类名哈希（`XXX_5-120-1` 模式）
- 兼容 ScriptCat 和 Tampermonkey
